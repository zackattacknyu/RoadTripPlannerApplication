There are two ways to launch the application:

- Double click on the jar file. 

- Go to a command prompt and do the following: 
	1. cd into the directory containing the jar file in this folder
	2. type in the following command:
		java -jar RoadTripPlanner_1.0.jar


When it is launched, do the following:
1. Open up the text file containing a list of stops
2. Decide where the excel file containing the data will be saved to
3. Decide any options, which are described in the window
